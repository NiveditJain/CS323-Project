To run the code,
Step 1 : install Python3
Step 2 : install dependencies using pip install -r requirements.txt or pip3 install -r requirements.txt
Step 3 : Go to Code Folder and choose the code you want to run by changing Folder
Step 4 : Make necessary changes to input files (see Code Presentation Video)
Step 5 : run python solver.py < input.txt (windows) or python3 solver.py < input.txt (macOS or Linux), for solver folder,
for generator folder run python generate_map.py < map_input.txt (windows) or python3 generate_map.py < map_input.txt (macOS or Linux)
